This crate implements an interface/glue layer between our ethernet drivers and the smoltcp network stack.

Structs

EthernetDevice

An implementation of smoltcp’s Device trait, which enables smoltcp to use our existing ethernet driver. An instance of this EthernetDevice can be used in smoltcp’s EthernetInterface.

EthernetNetworkInterface

A struct that implements the NetworkInterface trait for a NIC. There should be one instance of this struct per interface, i.e., an Ethernet port on the NIC.

RxToken

The receive token type used by smoltcp, which contains only a ReceivedFrame to be consumed later.

TxToken

The transmit token type used by smoltcp, which contains only a reference to the relevant NIC because the actual transmit buffer is allocated lazily only when it needs to be consumed.
