Expand description

A trait that defines the necessary minimum functions that all network interface card (NIC) drivers should implement.

Required methods

fn send_packet(
    &mut self,
    transmit_buffer: TransmitBuffer
) -> Result<(), &'static str>[src]

Sends a packet contained in the given transmit_buffer out through this NetworkInterfaceCard. Blocks until the packet has been successfully sent by the networking card hardware.

fn get_received_frame(&mut self) -> Option<ReceivedFrame>[src]

Returns the earliest ReceivedFrame, which is essentially a list of ReceiveBuffers that each contain an individual piece of the frame.

fn poll_receive(&mut self) -> Result<(), &'static str>[src]

Poll the NIC for received frames. Can be used as an alternative to interrupts, or as a supplement to interrupts.

fn mac_address(&self) -> [u8; 6][src]

Returns the MAC address that this NIC is configured with. If spoofed, it will return the spoofed MAC address, otherwise it will return the regular MAC address defined by the NIC hardware.
