Support for the Programmable Interval Timer (PIT) system clock, which enables and configures the frequency of simple timer interrupts and basic timer-based one-shot wait period.

pit_wait

Wait a certain number of microseconds, max 55555 microseconds. Uses a separate PIT clock channel, so it doesn’t affect the regular PIT interrupts on PIT channel 0.
